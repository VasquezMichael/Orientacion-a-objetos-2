# Orientación a Objetos 2 (OO2) — UNLP

Repositorio de trabajos prácticos, ejercicios y material de apoyo para la materia **Orientación a Objetos 2** (Universidad Nacional de La Plata).

## Objetivo
Centralizar el código fuente y el seguimiento de los ejercicios/prácticas de la cursada, manteniendo un historial de cambios ordenado con Git.

---

## Estructura del proyecto

- `src/`  
  Código fuente (Java).

- `pom.xml`  
  Configuración del proyecto con **Maven** (dependencias, build, plugins, etc.).

- `target/`  
  Carpeta generada por Maven al compilar (no se versiona).

---

## Requisitos
- **Java** (JDK instalado)
- **Maven**
- IDE recomendado: **Eclipse** o **IntelliJ**

> Si usás Eclipse, podés importar el proyecto como **Existing Maven Project** seleccionando la carpeta raíz del repo (la que contiene `pom.xml`).
